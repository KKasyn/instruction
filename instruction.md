## Списки
Для создания ненумеровонного списка необходимо пункты выделить (*). Вот так:
* элемент 1
* элемент 2
* элемент 3

для создания нумерованнных списков необходимо пункты пронумировать. Вот так:
1. Пункт
2. Пункт

Для выделения списк можно использовать символ (>). Вот так:
>1. Пункт
>2. пункт
